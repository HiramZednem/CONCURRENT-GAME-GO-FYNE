Este directorio contiene archivos Go para cada escena o pantalla del juego. 

Ejemplos:
    1. Escenas para el menu principal, 
    2. Escena para el juego en sí, y 
    3. Escena para la pantalla de fin de juego. 

En cada archivo de escena, puedes definir la lógica y la gestión de eventos específicos de esa escena.
